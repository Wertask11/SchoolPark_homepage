How to Start

3つのはじめかた

ウォレットなしで今すぐ試せます。準備ができたら、NFTを手に入れてフル機能を解放しましょう。

01
PAID

有料NFTユーザー

HEXAまたはOpenSeaにて有料NFTを購入し、ウォレットに入れた状態でログイン。すべての機能を制限なく利用できます。

02
FREE

無料NFTユーザー

ログインボタンから無料NFTを受け取り。ガス代はPolygonメインネット採用のため超格安。気軽に本格利用が可能。

03
GUEST

ゲストユーザー

ウォレット接続もNFTも不要。今すぐ気軽に入れます。投稿機能を中心にEmuの雰囲気をまず体験してみましょう。

Features

Emuでできること

知識の共有から、トークン獲得、学び、遊び、そして出会いまで。すべてがひとつのプラットフォームに。

📝

自由投稿 & テーマ投稿

自由に投稿するか、テーマ（本・失敗・経験・出会い）に沿った投稿で知識を共有。GoodとChangeで評価し合う。

🎓

探索・学び

第一線の専門家へのインタビューを要点・本文・補足でコンテンツ化。学校では教えてくれないことを深く学ぶ。

⭐

探索・遊び（星）

自分の活動や行動で星が生まれる。夜空に浮かぶ自分の星を眺めたり、次世代の履歴書として活用したり。

💬

探索・交流（議論）

１日１議論。ランダムで表示されるテーマをもとに、みんなでわいわい議論しよう。

🎮

探索・イベント

ECHO FIELD、秘密のとびら、林檎の方程式など。毎週水曜日に新イベントが追加される。

🔍

スマートフィルター

新規・Good・Change・おすすめ・トレンドのフィルターで、今の自分に必要な情報へ素早くアクセス。

EMUER
ERC-20 TOKEN

Token

活動が、
価値になる。

Emu独自のERC-20トークン「EMUER」は、あなたの活動や行動によって自然と蓄積されます。集めたEMUERはEmu内取引所でNFTと交換可能。

  • ログインボーナスで毎日獲得
  • 自分の投稿へのGood・Changeの数に応じて獲得
  • イベントやキャンペーンへの参加で獲得
  • Emu内取引所でNFTと交換可能

About

SchoolParkとは

学校のような、公園のような場所を作りたい——そんな想いから生まれたSchoolPark。国籍・学歴・性別・年齢、何にも縛られない分散教育の確立を目指します。

オンライン×オフラインを融合。世の中にSchoolParkを確立して、解散！
